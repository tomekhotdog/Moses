# AoC 2023 Day 1 — "Trebuchet?!"

Each line of text contains digits. The calibration value of a line is the
two-digit number formed by its FIRST digit and its LAST digit.

**Part 1:** Sum the calibration values across all lines.

**Part 2:** Spelled-out words ALSO count as digits:

    one, two, three, four, five, six, seven, eight, nine

Note that these can overlap (e.g. "eightwo" contains both "eight" and "two",
and "twone" contains both "two" and "one"). Recompute the sum with these
spelled-out words counting as digits.
